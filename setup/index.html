gitbook教程

gitbook 环境安装

node.js安装

到官网首页下载node.js,然后安装

gitbook安装

在命令行中使用node.js自带的npm安装工具，输入npm install -g gitbook-cli

npm install -g gitbook-cli

gitbook的扩展安装

不适用下面的功能可以不安装 用于转换为pdf的模块

npm install -g gitbook-pdf
